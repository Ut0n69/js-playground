This is a playground for JavaScript, TypeScript, and CSS.

## Feature

- webpack
  - HtmlWebpackPlugin
  - babel
- webpack-dev-server
- TypeScript
- ESLint
  - Prettier
  - typescript-eslint
- stylelint

## Usage

```
$ git clone https://github.com/Ut0n69/js-playground.git
$ cd js-playground/
$ yarn
$ yarn dev
```
